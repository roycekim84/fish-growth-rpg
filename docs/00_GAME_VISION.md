# Fish Adventure RPG

# Game Vision

Version: 1.0
Status: Draft

## Vision
Fish Adventure RPG is an exploration-focused RPG where the player starts as a tiny fish, discovers new species, unlocks new traversal abilities, and explores an ever-expanding aquatic world.

## Core Principles
- Exploration comes first.
- Hunting is a means, not the goal.
- Species represent new play styles.
- Every new biome should reward curiosity.
- GPS and real-world exploration are optional bonus progression.

## Core Fantasy
The player should constantly wonder: 'What is beyond that waterfall?' and 'If I unlock that species, where can I go next?'

## Success Criteria
- Exploration is exciting.
- Unlocking a new species changes gameplay.
- Players want to complete the encyclopedia.
- The world feels alive and worth revisiting.
